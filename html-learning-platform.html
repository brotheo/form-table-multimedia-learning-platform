üìñ What is CSS Flexbox?

Flexbox is a powerful layout tool that helps you align and distribute items in a container, even when their size is unknown. Perfect for responsive designs!

  • display: flex - Creates a flex container
  • justify-content - Aligns items horizontally (main axis)
  • align-items - Aligns items vertically (cross axis)
  • flex-direction - Sets direction (row, column)
  • flex-wrap - Allows items to wrap to next line
  • gap - Adds space between flex items

üìñ What is CSS Grid?

CSS Grid is a powerful two-dimensional layout system that lets you create complex web layouts with rows and columns. Unlike Flexbox (which is one-dimensional), Grid works on both axes simultaneously, making it perfect for organizing entire page structures!

  • display: grid - Creates a grid container
  • grid-template-columns - Defines column sizes (e.g., 1fr 2fr 1fr)
  • grid-template-rows - Defines row sizes
  • gap - Space between grid items (replaces grid-gap)
  • grid-column - Makes an item span multiple columns
  • grid-row - Makes an item span multiple rows
  • grid-template-areas - Creates named grid areas for semantic layouts
  • repeat() - Repeats pattern (e.g., repeat(3, 1fr) = three equal columns)
  • minmax() - Sets min and max size (e.g., minmax(200px, 1fr))
